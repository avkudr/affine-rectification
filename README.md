# affine-rectification
Affine rectification for SEM images 

## Installation

```
git clone https://github.com/avkudr/affine-rectification.git
cd affine-rectification
mkdir build
cd build 
cmake ..
make
```

## Testing

Launch ```./main_rectification```.
